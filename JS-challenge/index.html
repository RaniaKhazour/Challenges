Challenge Part 1

Task:

Create a program that takes a list of any number of strings and will determine which words in the list are exact anagrams of each other (ignoring spaces).

For example, if the input to the program is:

  • ['rope', 'pore', 'repo', 'red rum', 'murder', 'listen', 'silent', 'endeavour']
The output should be:
  • [['rope', 'pore', 'repo'], ['red rum', 'murder'], ['listen', 'silent'], ['endeavour']]
